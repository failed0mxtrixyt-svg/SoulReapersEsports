# Soul Reapers Esports Website

A modern, professional esports team website inspired by FaZe Clan's design aesthetic. Built with vanilla HTML, CSS, and JavaScript for maximum performance and compatibility.

## Features

- **Modern Design**: Dark theme with orange accent colors matching professional esports aesthetics
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Hover effects, animations, and smooth transitions
- **Team Roster**: Dynamic player cards with stats and information
- **Achievements Section**: Showcase tournament wins and accomplishments
- **Shop Integration**: Merchandise store with add-to-cart functionality
- **Smooth Navigation**: Sticky header with mobile hamburger menu
- **Animations**: Parallax effects, fade-in animations, and particle effects
- **Loading Screen**: Professional loading animation
- **Social Integration**: Footer with social media links

## Sections

1. **Hero Section**: Eye-catching landing area with team branding
2. **Team Roster**: Player profiles with stats and roles
3. **Achievements**: Tournament victories and prize winnings
4. **Shop**: Team merchandise and apparel
5. **About**: Team information and statistics
6. **Footer**: Social links and contact information

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Modern styling with animations and gradients
- **JavaScript ES6+**: Interactive functionality and animations
- **Google Fonts**: Oswald and Inter for typography
- **Font Awesome**: Icon library for social links and UI elements

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. No build process required - it's ready to run!

## Customization

### Colors
The main color scheme uses:
- Primary: `#ff6b35` (Orange)
- Secondary: `#f7931e` (Yellow-Orange)
- Background: `#000` (Black)
- Text: `#fff` (White)

### Fonts
- **Oswald**: Used for headings and branding
- **Inter**: Used for body text and content

### Images
Replace placeholder images with your own:
- Team photos in the roster section
- Product images in the shop
- Team photo in the about section
